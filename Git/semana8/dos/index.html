/*Contenedor Grid: Se define utilizando la propiedad display: grid; en CSS. Esto convierte un elemento en un contenedor Grid. Filas y columnas: Puedes definir el número y tamaño de filas y columnas con propiedades como grid-template-rows y grid-template-columns. Colocación de elementos: Los elementos hijos del contenedor Grid se colocan automáticamente o pueden ser posicionados manualmente con propiedades como grid-column y grid-row.*/
