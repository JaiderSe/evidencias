Explicación de Etiquetas HTML

Este documento explica algunas de las etiquetas HTML más comunes.

<!DOCTYPE html>

Declara el tipo de documento y la versión de HTML que se está utilizando.

<html>

La etiqueta <html> es el contenedor principal de un documento HTML.

<head>

La etiqueta <head> contiene metadatos sobre el documento, como el título y enlaces a hojas de estilo.

<meta charset="UTF-8">

Especifica la codificación de caracteres del documento, en este caso UTF-8.

<meta name="viewport" content="width=device-width, initial-scale=1.0">

Configura la vista del documento para que sea responsiva en diferentes dispositivos.

<title>

Define el título del documento que aparece en la pestaña del navegador.

<body>

La etiqueta <body> contiene el contenido visible de la página web.
