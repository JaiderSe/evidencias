Informe html listas y tablas

¿cómo crear una lista en ordenada html?

Para crear una lista en HTML, se utilizan las etiquetas <ol> para listas ordenadas y <ul> para listas no ordenadas. Dentro de estas etiquetas, se utilizan las etiquetas <li> para cada elemento de la lista.

  1. Primero
  2. Segundo
  3. Tercero

¿cómo crear una lista desordenada en html?

Para crear una lista desordenada en HTML, se utilizan las etiquetas <ul> y <li>. Dentro de la etiqueta <ul>, se utilizan las etiquetas <li> para cada elemento de la lista.

  • Manzana
  • Banana
  • Cereza

¿cómo crear una tabla en html?

Para crear una tabla en HTML, se utilizan las etiquetas <table> para definir la tabla, <tr> para definir una fila, <th> para definir una celda de encabezado y <td> para definir una celda de datos. Dentro de estas etiquetas, se pueden agregar los elementos de la tabla.
